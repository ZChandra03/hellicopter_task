┌──────────────────────────────────────────────────────────────────────────────┐
│ 🎯 KEY OUTPUTS EXPLAINED                                                     │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  σ_belief (fitted noise belief)                                             │
│  ├─ < true_σ  →  Overconfident   (underestimates noise)                    │
│  ├─ ≈ true_σ  →  Accurate        (correct noise model)                     │
│  └─ > true_σ  →  Underconfident  (overestimates noise)                     │
│                                                                              │
│  bias (fitted prior bias)                                                   │
│  ├─ < 0  →  Stay-biased   (expects fewer switches)                          │
│  ├─ ≈ 0  →  Unbiased      (no prior expectations)                           │
│  └─ > 0  →  Switch-biased (expects more switches)                           │
│                                                                              │
│  Cohen's κ (agreement)                                                      │
│  ├─ > 0.8  →  Excellent  (GRU ≈ Bayesian)                                   │
│  ├─ 0.6-0.8  →  Good     (mostly agree)                                     │
│  └─ < 0.6  →  Poor       (different strategies)                             │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
